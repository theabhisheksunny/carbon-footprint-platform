# Installation Note

## NPM Dependencies

Due to network connectivity, you'll need to run:

```bash
npm install express cors uuid
```

When you have network access. The package.json is already configured correctly.

## Required Packages (3 total)
- express@^4.18.2
- cors@^2.8.5  
- uuid@^9.0.0
