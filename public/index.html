🔌 API Endpoints

Our RESTful API makes it easy to integrate carbon tracking into your applications. All endpoints return JSON and use standard HTTP methods.

POST /api/users

Create a new user and calculate baseline footprint

GET /api/users/:userId/dashboard

Get complete dashboard with stats and recommendations

POST /api/activities

Log a carbon-generating activity

GET /api/actions

Browse available reduction actions

POST /api/calculator/compare

Compare CO2 emissions between two scenarios

GET /api/social/leaderboard

View community leaderboard

🚀 Quick Start

Get started in minutes. No database setup required!

1. Install Dependencies

npm install

2. Start the Server

npm start

3. Run the Demo

./demo.sh

4. Create a User

curl -X POST http://localhost:3000/api/users \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Jane Doe",
    "email": "jane@example.com",
    "location": "US",
    "profile": {
      "carMilesPerYear": 10000,
      "dietType": "vegetarian"
    }
  }'
